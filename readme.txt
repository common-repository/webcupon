== Description ==

Este plugin de WebCupón muestra un widget en su WordPress para ofertar Cupones de Descuento en línea, previa suscripción mensual, semestral o anual en la página oficial de WebCupón. Una vez se haya suscrito, podrá obtener desde su zona de cliente el ID API que tendrá que insertar en la configuración de este plugin para poder mostrar sus propios descuentos.

A parte de ofrecer sus ofertas a través del plugín, las mismas ofertas también serán publicadas en la página oficial ubicada en www.webcupon.es, en una extensión de Google Chrome, en Google Map y en aquellas webs que instalen este plugin de forma anónima y gratuita. Además, para aquellas regiones donde WebCupón tenga consolidado acuerdos publicitarios, se realizarán campañas  de publicidad indirecta en cadenas de Radio y Televisión local.

El plugin adiccionalmente crea dos páginas en tu WordPress llamadas "Directorio de Ofertas" y "Mapa de Ofertas", mostrando en un iframe mediante shortcode en cada una de ellas, los resultados de las Ofertas en vigor desde el servidor de WebCupón. También puedes insertar el código shortcode en "entradas" específicas si así lo deseas. Los resultados, tanto en el directorio como en el mapa, están constantemente actualizados.

El plugin conecta por iframe a páginas de WebCupón específicas para WordPress y a un mapa de Google Map que muestra todas las ofertas de descuentos vigentes. El objeto principal en este caso, sería el de obtener contenido para su propia web a través de WordPress.


== Installation ==

1. Instale 'webcupon' en el directorio '/wp-content/plugins/'
2. Active el plugin desde el subpanel 'Plugins' de WordPress
3. En el menú principal diríjase a Web Cupón e inserte el ID API

== Frequently Asked Questions ==

= ¿Realmente necesito un ID API? =

Sí. Necesita un código API que puede obtener desde [Web Cupón API](http://comercial.webcupon.es).

= ¿Este plugin funciona con WordPress Multisite? =

¡Por supuesto! El plugin WebCupón se ha probado y verificado, y funciona con la última versión de WordPress Multisite.

= ¿Puedo instalar WebCupón solo para mostrar ofertas? =

¡Sí! Usted puede instalar WebCupón para mostrar solamente las ofertas actualizadas al día de los Cupones de Descuento disponibles.
